# TechFlow Task Manager

## Objetivo

Este projeto simula o desenvolvimento de um sistema de gerenciamento de tarefas para uma empresa de logística utilizando conceitos de Engenharia de Software e metodologias ágeis.

O sistema permite organizar tarefas, acompanhar atividades e demonstrar um fluxo de desenvolvimento utilizando controle de versão, testes automatizados e gerenciamento visual.

---

## Escopo do Projeto

O sistema possui uma implementação básica de gerenciamento de tarefas, permitindo:

- Criar tarefas;
- Listar tarefas cadastradas;
- Remover tarefas.

O projeto foi desenvolvido utilizando Python com uma estrutura simples baseada em programação orientada a objetos.

---

## Metodologia Ágil

Foi utilizado o conceito de Scrum/Kanban para organizar o desenvolvimento.

O acompanhamento das atividades foi realizado através do GitHub Projects, utilizando as seguintes etapas:

- A Fazer;
- Em Progresso;
- Concluído.

---

## Controle de Qualidade

Foi configurado um pipeline utilizando GitHub Actions para execução automática dos testes do sistema.

A cada alteração enviada ao repositório, os testes automatizados são executados para verificar se o código continua funcionando corretamente.

---

## Histórico de Desenvolvimento

O projeto foi desenvolvido utilizando Git com commits organizados para registrar cada etapa:

- Criação da estrutura inicial do projeto;
- Implementação do gerenciador de tarefas;
- Criação dos testes automatizados;
- Configuração do GitHub Actions;
- Atualização da documentação.

---

## Gestão de Mudanças

Durante o desenvolvimento do projeto, o cliente solicitou uma alteração no escopo inicial.

### Nova funcionalidade solicitada

Foi adicionada a necessidade de incluir níveis de prioridade nas tarefas:

- Alta;
- Média;
- Baixa.

Essa mudança foi registrada no backlog do projeto e adicionada ao quadro Kanban do GitHub Projects para acompanhamento da equipe.

A alteração demonstra a importância da adaptação em projetos ágeis, permitindo que o sistema evolua conforme as necessidades do cliente.
